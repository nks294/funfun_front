### Clone project (https)
```powershell
git clone https://github.com/nks294/funfun_funding.git
```

<br>

### Go to project (frontend)
```powershell
cd funfun_funding
```
### Install Package (frontend)
```powershell
npm install
```
### Start frontend
```powershell
npm start
```

<br>

### Go to project (backend)
```powershell
cd funfun_funding/backend
```
### Build project (backend)
```powershell
./gradlew clean build
```
### Start backend
```powershell
java -jar build/libs/backend.jar
```
